# Carousel

##CRUD Operations

HTTPVerb-> Endpoint	-> Action
POST    -> /api/carousel/:id   -> CREATE a new item into DB
GET	    -> /api/carousel/:id	 -> READ data and return data
PUT	    -> /api/carousel/:id	 -> UPDATE data on item
DELETE  -> /api/carousel/:id	 -> DELETE item in DB
